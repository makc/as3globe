AS3 Globe is simple FP9 globe component

Key features:
 - custom markers support
 - custom projection support (mercator, etc)
 - simple API (no math degree required)
 - minimal distortions (mesh optimized for equirectangular map)
 - small dedicated 3D engine (less than 8K footprint)
 - lazy rendering (does not burn your CPU)
